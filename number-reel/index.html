Number Reel

Number Reel Overview

To trigger the number reel just use data-attributes data-trigger="numberReel" or programatically el.numberReel.

The plugin then takes the content, which should be a number, and intiallizes the reel. The animation is handled by CSS.

Example

233242
<!-- the code -->
<div data-trigger="numberReel">233242</div>

Options

Name type default description
currency string '$' Prepend a currecy symbol or any string. '', prepends NULL
infinate string 'infinate' 'infinate' animates the numbers continuously with a small delay configurable in the CSS. '', animates only once.
reelFace number 9 appends X other numbers, in a random order, to use in the animation
seperator string 'thousand' 'thousand' animates the numbers continuously with a small delay configurable in the CSS. '', animates only once.
